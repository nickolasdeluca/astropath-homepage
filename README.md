# astropath-homepage

Source code for [astropath.com.br](https://astropath.com.br) — the personal developer page of **Nickolas de Luca Alberton**.

Built with [Astro](https://astro.build). Themed after the lore of Warhammer 40,000.

## Stack

- **Framework**: Astro (static output)
- **Styling**: Plain CSS with custom properties
- **Fonts**: Cinzel, Cinzel Decorative, Share Tech Mono (Google Fonts)

## Development

```bash
yarn
yarn dev
```

## Build

```bash
yarn build
```

Output is written to `dist/` and can be deployed to any static host.
